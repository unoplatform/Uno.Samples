# Vector Graphics Showcase

This sample demonstrates the vector graphics capabilities of Uno Platform 6.6 on the Skia rendering stack. It covers stroke caps and joins (`StrokeStartLineCap`, `StrokeEndLineCap`, `StrokeLineJoin`), dashed strokes (`StrokeDashArray`, `StrokeDashCap`), animatable `Geometry.Transform` driven by a `Storyboard`, and per-element alpha masks produced with `GetAlphaMask()` and the Composition `CompositionMaskBrush`.

The project targets Desktop and WebAssembly, both using the Skia renderer.
